About US

In this website you can find and learn all the core basic knowledge that you would need to start an online website. I will be covering from the history down all the way to JQuery. This will give you a taste of how the website work not just how you could build one. Learning about the concept about the website is much more important than you learning how to build a website because it will teach how each of the parts communicate with other parts in a website.

Use the navigation to navigate around this website, some of the parts are linked in other webpages that I have created. For example you can find some basic information about form in the HTML part and if you want to learn more about form then you can find a section in JavaScript which is basically an advanced level of using forms on a webpage.

I have built this website using Notepad ++ and Dreamweaver. In order to bring everything on to the screen neatly and interactive I have used CSS, Java Script as well as J Query for navigation purposes.
