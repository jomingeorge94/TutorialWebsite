Jquery Intro

JQuery is a a library for the JavaScript and it's purpose is to make it easier for people to use JavaScript on their website. It is free and an open source code which is been licensed under MIT and GNU. JQuery was released in January 2006 by John Resig at BarCamp NYC.

One of the main advantage of using JQuery is that it's been supported by most of the browsers includes Internet Explorer, Google Chrome and Firefox. In order to use JQuery you don't have to learn new syntax, you just have to know the basic syntax of using JavaScript. JQuery is simple and you only have to write couple of lines of codes instead of writing a big chunk of code. You have to make sure that you have got the correct version of JQuery as they update it all the time.

If you want to use JQuery on your website, either download or link it to the JQuery library website (Google CDN, Microsoft CDN, etc).
