Background Property

With the use of CSS Background you can change the effect of an elements background properties. These are some of the Background properties; background-color, background-image, background-repeat, background-position, etc.

Background Color

Backgroung Color property lets you change the background colour of an element. In this example you will see an example of changing the background colour of a website by using this property in the body slector and choosing a colour for the background.

Background Image

Background Image property allows you to select an image and choose than as your background of an element.

Box Model CSS

Pretty much all the elements are constructed of a box model where you could design and layout in CSS. Box Model basically means a wrap around the HTML element to change its properties such as margin, border, padding and the content. This image below shows the Box Model.

Border

Border goes around the padding and the content. It is been made up of three properties, which are; width, style and color.


Padding

Padding is the space between the content and the inside edge of its box.


Margin

Margin is the space around the outside edge of a box.


CSS Overflow

Overflow lets you control an element when it spill out of the box. Overflow is supported by most of all the browsers includes Internet Explorer, Google Chrome, Firefox, etc.
