Java Script Forms

In this page you will be learning about Regular Expressions that you could use on forms for validation purposes. Regular Expression is a condition that the user has to meet in order to go to the next object/element. For example, if you have a field where you want the user to type their email address and if someone writes their telephone number by mistake, then it's a problem of validating your form. However with the help of Regular Expression you can easily put a condition that the user has to meet for each of the elements/fields in a form.
