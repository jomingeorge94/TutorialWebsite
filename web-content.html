Web Content

All the Web Documents uses the Language called Hypertext Mark-up Language (HTML). When you submit your source code to a browser it has to process the document that you have given and then display the content to the user.

Usually a Web Document has three parts that they have to deliver;

  • Content
    • This is the basic HTML page you are structuring out which would have paragraphs, headings, images, etc. Note that it would not have any Styling at all.
  • Presentation/Styling
    • Now that you have done your webpage, to style your website you can use the Cascading Style Sheets (CSS) to change the layout, clour scheme, and graphic of the web page.
  • Behaviour
    • For interactive purpose, you might add in features which would be eye-catching to the users. Use JavaScript, JQuery, etc to perform this action. You can use this section to handle the events, respond to what ever the user clicks on, etc.
