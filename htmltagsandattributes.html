HTML - Tags

In this page, you will be learning about all the tags that you can use to deliver your website on to the Browser. Usually all the tags are wrapped inside one of these angle brackets <> and it would have same one at the end where it will close the angle bracket with a forward slash key </>.

HTML - Heading

HTML Headings has six levels from h1 to h6.

HTML - Paragraphs and Breaks

If you want to use the HTML paragraph use <p> tag. If you want to have a line break use it like this, <br>.

HTML - Images

If you want to add an iamge to your website, use the <img> tag. Now you can use attributes to do some more additional features such as an alt attribute which will display on the image area if the image cannot be displayed, title attribute is used to display a text when you hover over the image, and also you can use the height and width to give your picture an exact height and width.

HTML - Lists

There is three types of HTML lists, Unordered Lists, Ordered Lists and Description/Definition Lists

Unordered Lists

Unordered List is a bullet list, you can specify that its an unordered list by using the <ul> tag. After specifying that its an Unordered List, you can used this <li> tag to list out each of the items.

Ordered Lists

Ordered List is a numbered list which starts with a <ol> tag. After specifying that it's an ordered list, you can use the <li> tag to list out each of the items.

Description/Definition Lists

Description/Definition Lists is not commonly used list however it is used to create definition lists. You have to use the <dl> tag to specify that its an definition list. To list use the <dd> tag.

HTML - Internal and External Links

When making links, you have two options either you can link to another page in your site or to an external website. Use the , <a> tag to make links.

Internal Links

<a href="index.html">, this code is used to go to another page which is in your website. href specifies the exact file location of the page that you want to link.

External Links

If you want to link it an external site, do it same as you do it to an internal but this time, you just have to write the full url of the site that you want to link it to.
