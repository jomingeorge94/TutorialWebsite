HTML Form Tutorial

Checkboxes

Checkboxes are used on a form to let the user to select more than one option from a number of choices. This is a demo of how it would look like on a webpage;

Which of the following activities you are enrolled into? (you can select more than one)
Cricket
Football
Tennis
None of them

Drop Down Lists

Drop Down Lists are used in a web form to allow the users to select an option from a list.

What is your favourite colour?

Multi-Line Input

With the use of the Multi-line input the user will be able to fit more character than using a text field which will have a character limit usually around 50 words. This is a demo showing how it looks like on a webpage;

Describe yourself (hobbies, skills,etc)

Password Field

Password Field is used in web form for security purposes. It is exactly same as one-line input however it hides the input and display the text in a bulletpoint format.

Type your password:

Submit Button

Submit Button on a web form is used to submit the data on the form to a server or to an external link.
