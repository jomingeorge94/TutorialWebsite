HTML Form Tutorial

Forms are used on a webpage to allow the users to enter data which then will be passed to a server for analysing the data. Forms are usually done by either paper or online using radio button, text fields, check boxes, buttons, etc. In this section of the tutorial we will be only looking at the basic use of forms on a webpage (i.e. how to input text fields, password field, etc.), however later on in the JavaScript section we will be looking more advanced features that you could use on a form such as regular expressions, etc.

Form Control is not hard to understand, all you need to know is that each form control will have a name and sometimes a value attribute. Form has two methods, post and get methods. Post methods are used to send the data on a form to a server as a HTTP transaction wheras the Get method will show the data (name and value) on the URL. It is much more safer and secure to use the POST method instead of the GET method.

Input element is used to collect the user's information. It can vary depending on the attributes, these are some of the type which we will be covering in this webpage; text field, radio buttons, check boxes, drop down lists, multi-line input, password field and submit button.
